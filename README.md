# Сервис FlowerShop

Онлайн сервис Telegram-бота цветочного бизнеса для автоматизации заказов. 

## Запуск

Для запуска сервиса у вас уже должен быть установлен Python 3.8.9.

- Установите зависимости командой `pip install -r requirements.txt`
- Определите настройки доступа сервиса к Telegram-боту и каналам/пользователям (ниже есть полезная информация по этим темам). Для этого создайте файл `.env` рядом с `bot.py` и запишите туда данные в таком формате: `ПЕРЕМЕННАЯ=значение`.  
    Требуются следущие переменные:
    - `TG_BOT_TOKEN` — API-токен Telegram-бота.
    - `FLOURIST_TG_ID` — ссылку на канал флориста.
    - `COURIER_TG_ID` — ссылку на канал курьера.  
    Пример заполнения файла:
    ```
    TG_BOT_TOKEN=958423683:AAEAtJ5Lde5YYfkjergber
    FLOURIST_TG_ID=@example_channel
    COURIER_TG_ID=@example_channel
    ```
- Запустите приложение командой `python .\bot.py`. Если настройки в файле `.env` введены верно, бот готов к работе.

## Схема бота

Файл flowershop_flow_1.dravio содержит визуальную схему работы Telegram FlowerShop бота.

В схеме отражены основные сценарии:

- запуск бота
- согласие с политикой конфиденциальности
- главное меню
- подбор букета
- оформление заказа
- консультация флориста

Файл можно открыть в diagrams.net (Draw.io).
